TopoVessel: Branch-Level Coronary Artery Segmentation via Differentiable Skeleton

Anonymized Authors

Abstract

Precise branch-level coronary artery segmentation in X-ray fluoroscopy is critical for guiding Percutaneous Coronary Intervention (PCI) and enabling automated Quantitative Coronary Analysis (QCA). However, this task remains highly challenging due to low signal-to-noise ratios, extreme data scarcity, and failures in maintaining topological continuity in complex vascular trees.

We curate a proprietary dataset of 10,924 expert-annotated angiography frames with 21 anatomical subsegments — an order of magnitude larger than existing benchmarks. Architecturally, we introduce a hybrid dual-branch encoder that synergizes local CNN features with global DINOv3 context, coupled with a Differentiable Skeleton (DS) module that regularizes decoding using anatomically grounded extreme keypoints.

TopoVessel is the first framework to surpass 0.82 Dice score for branch-level coronary segmentation, while maintaining real-time performance at 40 FPS. Our method establishes a new standard for clinically viable, topologically consistent vascular analysis.

Method Overview

TopoVessel consists of:

  • Hybrid Dual-Branch Encoder — CNN local geometry + DINOv3 global context
  • SimCC-based Extreme Point Localization — classification-based coordinate prediction
  • Differentiable Skeleton Module — pixel-level vector field enforcing global topology

Dataset

We construct a large-scale in-house X-ray angiography dataset with 10,924 annotated frames across seven clinical projections. The dataset includes 21 anatomical branches such as LAD, LCx, diagonals, marginals, septal perforators, and PDA.

  • Total Images: 10,924
  • Train: 9,832
  • Test: 1,092
  • Branches: 21 anatomical subsegments

Results

TopoVessel achieves:

  • Branch-level Dice: > 0.82
  • Improved Worst-Branch Performance
  • Real-time Throughput: 40 FPS
